Compile the Code: Ensure you have Java installed. Navigate to the project directory in your terminal and compile the code using:
    javac *.java

Run the Application: Start the application with:
    java AstronautScheduleApp


Interacting with the Application:
    You will see a menu with options to add, remove, view tasks, mark tasks as completed, and filter tasks by priority.
    Follow the prompts to enter task details, such as description, start time, end time, and priority.
    To exit the application, select the option to exit (option 6).


Note on Time Format
    The application expects the time to be entered in HH format (e.g., 07:00 for 7 AM). Ensure that the entered time adheres to this format to avoid errors.

Error Handling
    The application provides error messages for invalid time formats, task conflicts, and attempts to remove non-existent tasks.
